# Lógica para Computação

Repositório dedicado ao suporte pedagógico das disciplinas de **Lógica para Computação**. Este espaço centraliza ferramentas interativas, simuladores e materiais didáticos desenvolvidos para facilitar o aprendizado de Lógica de Predicados e Proposicional.

## App: Analisador de Árvore Sintática Interativo

O projeto atual deste repositório é uma ferramenta interativa desenvolvida para auxiliar estudantes na compreensão da estrutura gramatical de fórmulas lógicas.

### Principais Funcionalidades
- **Parser Robustos:** Processamento de Lógica de Predicados e Proposicional.
- **Visualização Hierárquica:** Geração dinâmica de árvores sintáticas baseadas em SVG e HTML.
- **Interatividade:** Seleção de subfórmulas via *hover*, permitindo visualizar escopos, variáveis livres e presas.
- **Passo a Passo (Wizard):** Modos pedagógicos *Top-Down* (raiz para folhas) e *Bottom-Up* (folhas para raiz).
- **Exportação:** Opção de salvar a árvore gerada em formato PNG de alta resolução.
- **Autoajuste:** Ferramentas de zoom e expansão para visualização de árvores complexas.

### Como utilizar
Como este é um projeto baseado em tecnologias Web (HTML/JS), não é necessária uma instalação complexa.

1. **Clone o repositório** ou faça o **Download** dos arquivos.
2. Abra a pasta do projeto.
3. Clique duas vezes no arquivo `index.html` (ou o nome do arquivo principal) para abri-lo no seu navegador padrão (Google Chrome, Firefox ou Edge são recomendados).
